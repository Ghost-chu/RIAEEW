# RIAEEW 

RIAEEW 基于 GeoIP 技术，为玩家提供地震提前预警功能，支持读秒和更新报



## IP 库

需要使用 [IP2LOCATION](https://www.ip2location.com/)，IPV4 文件命名为 `IP2LOCATION-IPV4.BIN`, IPV6 文件命名为 `IP2LOCATION-IPV6.BIN` 放进数据目录。
